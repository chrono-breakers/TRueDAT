# TRueDAT
BAT scripts for *dongleless* DJMAX TRILOGY which allow you to easily backup and restore multiple saves

obviously, this only works on Windows.

# setup
just copy the contents of the repository into the root of your game, INCLUDING the `TR_SAVE` folder.

you may delete `dummy`

# usage
double click the scripts

``Savedata-BackUp.bat`` can create slots, and back up your current data to save slots

``Savedata-Restore.bat`` restores the savedata from a slot
